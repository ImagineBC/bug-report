To publish sessions to the shared dashboard, you'll need a GitHub Personal Access Token (PAT). Without one, you can still use the form locally — drafts auto-save and you can download PDFs anytime — they just won't appear on the shared dashboard.

GitHub PAT

  1. Open GitHub's PAT page (pre-filled)
  2. Under Repository access → Only select repositories → pick ImagineBC/bug-report
  3. Under Permissions → Repository permissions → set Contents to Read and write
  4. Click Generate token, copy it (starts with github_pat_…)
  5. Paste below
